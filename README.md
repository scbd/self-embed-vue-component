# self-embed-vue-component

## Project setup
```
yarn install
```
### Build dependant dev widget for serving
```
yarn run build:dev:widget
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
